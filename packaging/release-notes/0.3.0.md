# Entra PIM Manager 0.3.0

Choose how the app integrates with Windows the first time you run it.

## New

- **First-run setup.** The first time you launch the app after installing, a one-time dialog lets you choose whether Entra PIM Manager should **start with Windows** and whether to keep its **Start menu entry**. Both are on by default, so the recommended setup is a single click on "Get started".
- **Start menu entry toggle.** The new choice is also available afterwards under **Settings → Behavior**, alongside the existing "Start with Windows" toggle — remove or restore the Start menu shortcut at any time.

## Fixed

- **Start menu opt-out now works.** Declining the Start menu entry during first-run setup — or turning the toggle off in Settings — left the shortcut in place because the app looked for it in the wrong location. The shortcut is now created, removed, and checked at the correct path, and an entry left behind by an earlier version is cleaned up automatically.

## Notes

- The setup dialog appears only once per installation. Both options remain changeable in Settings later.
