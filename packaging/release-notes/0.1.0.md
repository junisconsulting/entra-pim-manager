# Entra PIM Manager 0.1.0

First public release.

## Features

- Tray-based activation of Microsoft Entra PIM eligibilities — Directory Roles, Group Memberships, and Group Ownerships in one list
- Multi-tenant support: sign in with multiple admin accounts and see eligibilities and active assignments grouped per tenant
- WAM-broker authentication via MSAL — no embedded WebView, no password prompts inside the app
- Device-code sign-in fallback (Add account → Advanced) for tenants whose federated IdP forces seamless SSO onto the wrong account — complete sign-in on a second device
- Activation panel with justification, ticket reference, and duration slider (0.5 h steps, up to the per-role maximum)
- Live activation watchdog that auto-refreshes the list when assignments are activated, deactivated, or expire
- Favorites for recurring justifications, persisted per user
- Drag-and-drop reordering of accounts in Settings
- Per-user install to `%LocalAppData%\Programs\Entra-PIM-Manager\` — no UAC, no HKLM writes
- Optional Windows autostart (enabled by default on first install, toggleable in Settings)
- Velopack auto-update channel for in-place upgrades
